Guía de controles

⬆️ or = or + → suma +1
⬇️ or - → resta -1
0 → resetea el valor actual a 0
Home → bloquea / desbloquea el control
🔒 / 🔓 → hace lo mismo que Home
Meta final → cambia el número después de la barra
Texto → cambia la palabra final
Valor actual → escribe un número exacto manualmente
Fuente → cambia solo la tipografía del overlay
Apariencia → cambia colores y fondo del overlay
reset de Meta final → vuelve a 10
reset de Texto → vuelve a wins
reset de Valor actual → vuelve a 0
Cuando el control está bloqueado, ni las teclas ni los botones cambian nada.
